# vue-form-validation

## Project setup
```
npm install
```
dependencies:
- vuex
- vue-router
- vuelidate
- nprogress


### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

- Sign up
